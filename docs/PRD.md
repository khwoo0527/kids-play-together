# PRD: 풍선 터트리기 터치 게임

> 버전: 1.0 | 작성일: 2026-03-17

---

## 1. 제품 개요

### 목적
초등학생이 수업 시간 또는 쉬는 시간에 터치 모니터/태블릿/스마트폰 브라우저에서 즐길 수 있는 풍선 터트리기 게임. URL 공유만으로 별도 설치 없이 접속 가능하다.

### 핵심 컨셉
- 풍선이 화면 **아래에서 위로** 올라온다 (아이들 손이 닿는 하단 영역)
- 화면 **절반을 넘으면 자동으로 터짐** (점수 없음, 패널티 없음)
- 터치 한 번으로 즉시 터트리기 — 조작이 단순

---

## 2. 사용자 페르소나

| 페르소나 | 설명 |
|----------|------|
| 학생 (주 사용자) | 6~13세 초등학생. 스마트폰/태블릿 터치에 익숙. 긴 설명 없이 바로 플레이 원함 |
| 교사 (보조 사용자) | URL을 학생들에게 공유. 별도 설정 없이 수업에 활용 가능해야 함 |

## 3. 기능 요구사항 (MoSCoW)

### Must Have
- 풍선 렌더링 및 애니메이션 (아래 → 위 이동, 좌우 흔들림)
- 터치/클릭으로 풍선 터트리기
- 절반 이상 올라간 풍선 자동 터짐 (점수 없음)
- 점수 시스템 (풍선 1개 = 10점, 콤보 배율)
- 타이머 (30초 / 60초 / 무제한 선택)
- 3초 카운트다운 후 게임 시작
- 랭킹 저장/표시 (localStorage, 상위 10개)
- 시작 화면 / 게임 화면 / 결과 화면
- 이모지 아바타 선택 (6종)
- 전체화면 (브라우저 window 크기 꽉 채움)

### Should Have
- 귀여운 효과음 (팝 소리, 콤보 소리, 게임 종료 소리)
- 배경 멜로디 (Web Audio API 합성음)
- 다양한 풍선 색상/크기
- 콤보 시스템 + "대박!/최고!" 텍스트 이펙트
- 풍선 터질 때 파티클 이펙트
- 폭탄 💣 풍선 (터치 시 점수 -20)
- 보너스 ⭐ 풍선 (터치 시 점수 +30)

### Could Have
- 배경 테마 변경
- 난이도 선택 (풍선 속도 조절)
- 풀스크린 API 지원

### Won't Have (v1)
- 서버 기반 글로벌 랭킹
- 멀티플레이어
- 계정/로그인
- 닉네임 텍스트 입력 (이모지 아바타로 대체)

> 📌 **미래 계획**: v2 이후에 다양한 게임 버전 추가 예정 (미로, 숫자 맞추기 등). 공통 랭킹/사운드 모듈 재사용 가능하도록 설계.

## 5. UX 원칙 (초등학생 대상)

- 밝은 원색 풍선, 큰 글씨, 큰 터치 영역
- 실패 패널티 없음 (폭탄 풍선만 감점)
- 결과 화면에서 항상 칭찬 메시지
- 3초 카운트다운으로 준비 시간 제공
- 콤보 달성 시 이펙트로 흥미 유발

## 7. 화면 구성

```
시작 화면
  → 아바타 선택 + 시간 선택 + 게임 시작 버튼
  → 랭킹 보기 버튼

3초 카운트다운

게임 플레이
  → HUD (점수 / 타이머 / 콤보)
  → 절반선 (위험 구역 표시)
  → 풍선 올라옴 + 터치로 터트리기

결과 화면
  → 칭찬 메시지 + 최종 점수
  → 통계 (풍선 수 / 최고 콤보 / 정확도)
  → 랭킹 Top 10
  → 다시하기 / 처음으로
```
